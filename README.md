Still Patterns is a calm, contemplative web application designed to support reflection, pattern recognition, and clearer decision-making.

The project hosts time-tested symbolic practices in a quiet, focused digital environment. Rather than offering prediction or certainty, it encourages users to slow down, notice structure, and engage thoughtfully with their own questions.

This repository currently features an interactive I Ching (Book of Changes) experience built with modern web technologies and a deliberately restrained design philosophy.

Design Philosophy

Still Patterns is intentionally minimal.

No accounts

No notifications

No gamification

No predictive claims

The app is designed as a reflective tool, not an authority. Symbolic output is presented as a lens for thought rather than an answer to be obeyed.
